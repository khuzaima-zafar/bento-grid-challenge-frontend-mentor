### What I Learned

While building this project, I improved my understanding of:

* Creating complex layouts using **CSS Grid**
* Using **grid-column** and **grid-row** for spanning elements
* Structuring layouts with different card sizes
* Making layouts responsive for smaller screens

Example of grid setup:

```css
.container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 20px;
}
```
